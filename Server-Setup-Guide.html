ติดตั้ง Node.js

ติดตั้งและตั้งค่า PM2
รัน Process ตลอด 24 ชั่วโมง และ auto-start เมื่อ reboot
REQUIRED 
# ติดตั้ง PM2 แบบ global
sudo npm install -g pm2

# รัน Bot Server
pm2 start server.js --name "ehp-bot"

# รัน Ticket API
pm2 start ticket-api.js --name "ehp-ticket-api"

# ตั้งให้ start อัตโนมัติเมื่อ reboot
pm2 startup
pm2 save

# ดูสถานะ
pm2 status
pm2 logs ehp-bot

ติดตั้ง Nginx + SSL Certificate
LINE บังคับใช้ HTTPS — ต้องมี Domain Name
REQUIRED 
# ติดตั้ง Nginx และ Certbot
sudo apt install -y nginx certbot python3-certbot-nginx

# ขอ SSL Certificate ฟรีจาก Let's Encrypt
sudo certbot --nginx -d your-domain.com

สร้างไฟล์ Config Nginx:

sudo nano /etc/nginx/sites-available/ehp-bot
# วางเนื้อหานี้ในไฟล์
server {
    listen 443 ssl;
    server_name your-domain.com;

    location /webhook/line {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
    }
    location /notify/ {
        proxy_pass http://localhost:3000;
    }
    location / {
        proxy_pass http://localhost:4000;
    }
}
# เปิดใช้งาน Config
sudo ln -s /etc/nginx/sites-available/ehp-bot /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx

ตั้งค่า IIS Reverse Proxy + SSL
LINE บังคับใช้ HTTPS — ต้องมี Domain Name
REQUIRED
💡 ต้องติดตั้ง IIS + Application Request Routing (ARR) + URL Rewrite Module ก่อน

1. เปิด IIS Manager → Sites → Add Website

2. ตั้ง Binding: HTTPS port 443 พร้อม SSL Certificate

3. เพิ่ม web.config ในโฟลเดอร์ IIS site:

<configuration>
  <system.webServer>
    <rewrite>
      <rules>
        <rule name="webhook">
          <match url="^(webhook|notify)/(.*)" />
          <action type="Rewrite"
            url="http://localhost:3000/{R:0}" />
        </rule>
        <rule name="api">
          <match url="^(.*)" />
          <action type="Rewrite"
            url="http://localhost:4000/{R:0}" />
        </rule>
      </rules>
    </rewrite>
  </system.webServer>
</configuration>
⚠️ SSL Certificate สามารถขอจาก Let's Encrypt (win-acme) ฟรี หรือซื้อจาก CA ทั่วไป
