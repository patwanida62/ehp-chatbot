ติดตั้ง Node.js

ต้องการ Node.js v18 ขึ้นไป

▼

ดาวน์โหลด

🌐 ไปที่ https://nodejs.org → เลือกดาวน์โหลดเวอร์ชัน LTS (แนะนำ) → ติดตั้งตามขั้นตอน

ตรวจสอบเวอร์ชัน

node --version    # ต้องได้ v18.x.x ขึ้นไป
npm  --version    # ต้องได้ v9.x.x ขึ้นไป

▼
⚠️ ต้องติดตั้ง Git ก่อน — ดาวน์โหลดได้ที่ https://git-scm.com

4

สร้างไฟล์ .env

ตั้งค่า LINE Token, Secret และ URL

▼
🔒 ไฟล์ .env เก็บข้อมูลลับ ห้าม commit ขึ้น GitHub เด็ดขาด

สร้างไฟล์ .env ในโฟลเดอร์ ehp-chatbot

LINE_CHANNEL_ACCESS_TOKEN=ใส่ Token จาก LINE Developers Console
LINE_CHANNEL_SECRET=ใส่ Channel Secret จาก LINE Developers Console
TICKET_API_BASE_URL=http://localhost:4000
PORT=3000

หา Token และ Secret ได้ที่ไหน

  • 1️⃣ ไปที่ developers.line.biz → เลือก Provider → เลือก Channel
  • 2️⃣ แท็บ Basic settings → คัดลอก Channel secret
  • 3️⃣ แท็บ Messaging API → คัดลอก Channel access token
5

รัน Server ทั้ง 2 ตัว

เปิด Terminal 2 หน้าต่างพร้อมกัน

▼

Terminal 1 — Webhook Bot (port 3000)

server.js
$ npm run dev
Webhook server running on port 3000

Terminal 2 — Ticket API (port 4000)

ticket-api.js
$ npm run ticket-api:dev
Ticket API running on port 4000
✅ ถ้าเห็นข้อความ "running on port" ในทั้งสอง Terminal แสดงว่า Server ทำงานปกติ
6

เปิด ngrok (สำหรับ Dev/Test)

สร้าง HTTPS URL ชั่วคราวเพื่อรับ Webhook จาก LINE

▼
⚠️ ต้องดาวน์โหลด ngrok เวอร์ชัน 3.20.0+ จาก https://ngrok.com/download และทำการ login ก่อน

เพิ่ม Auth Token (ครั้งแรกครั้งเดียว)

ngrok config add-authtoken YOUR_NGROK_TOKEN

เปิด Tunnel ไปยัง port 3000

ngrok http 3000

ผลลัพธ์ที่ได้ (ตัวอย่าง)

Forwarding https://xxxx-xxxx.ngrok-free.app → http://localhost:3000
📋 คัดลอก URL ส่วน https://xxxx-xxxx.ngrok-free.app เพื่อใช้ในขั้นตอนถัดไป
7

ตั้งค่า Webhook ใน LINE Developers

เชื่อม LINE OA กับ Bot Server

▼
  • 1️⃣ ไปที่ developers.line.biz → เลือก Channel ของคุณ
  • 2️⃣ แท็บ Messaging API → หัวข้อ Webhook settings
  • 3️⃣ ใส่ Webhook URL: 
    https://xxxx-xxxx.ngrok-free.app/webhook/line
  • 4️⃣ กดปุ่ม Verify → ต้องได้ Success
  • 5️⃣ เปิด Use webhook: ON
⚠️ ใส่ URL ให้ครบถึง /webhook/line ด้วย ไม่ใช่แค่ domain เฉยๆ
8

ทดสอบระบบ

ตรวจสอบว่าทุกอย่างทำงานถูกต้อง

▼

ทดสอบผ่าน LINE

  • 💬 พิมพ์ใน LINE: "แจ้งปัญหา" หรือ "เข้าระบบไม่ได้" → Bot ควรตอบกลับพร้อม Ticket #
  • 🔍 พิมพ์: "ตรวจสอบ #1" → Bot ควรแสดงสถานะ Ticket

ตรวจสอบ Admin UI

  • 🌐 เปิด Browser ไปที่ http://localhost:4000/admin
  • ⚙️ แท็บ Keywords — จัดการคำที่ทำให้เปิด Ticket อัตโนมัติ
  • 📋 แท็บ Settings — ตั้งค่า Service / Category / SubCategory

ตรวจสอบ Logs ใน Terminal

[LINE] userId=Uxxxxxxxx | message="แจ้งปัญหา"
[TICKET] Auto-created #3 for "ชื่อผู้ใช้" keyword="แจ้งปัญหา"
🎉 ถ้าเห็น log เหล่านี้ใน Terminal แสดงว่าระบบทำงานสมบูรณ์แล้ว!

📌 สรุป URL และ Port ที่สำคัญ

บริการ URL Port สถานะ
Webhook Bot http://localhost:3000/webhook/line 3000 ต้องเปิดตลอด
Ticket API http://localhost:4000/tickets 4000 ต้องเปิดตลอด
Admin UI http://localhost:4000/admin 4000 เปิดเมื่อต้องการ
ngrok Tunnel https://xxxx.ngrok-free.app — เฉพาะ Dev/Test
